# Tech Flow Runner Game

## Overview
Tech Flow Runner is an engaging game that challenges players to navigate through a circuit board filled with various obstacles. The objective is to guide the Tech Flow Runner to complete levels while avoiding traps and enemies.

## Gameplay
- Players control the Tech Flow Runner using keyboard commands.
- The Runner can dash, jump, and slide to evade objects and enemies.
- There are multiple levels, each with unique challenges and a boss fight at the end.

## Features
- **Dynamic Obstacles:** Different types of obstacles that may move or change.
- **Power-Ups:** Obtain various power-ups to enhance abilities temporarily.
- **Progressive Difficulty:** Each level gets progressively harder with more obstacles and faster enemies.
- **Scoring System:** Players earn points based on performance, speed, and level completion.

## Controls
- **Arrow Keys:** Move left and right.
- **Space Bar:** Jump.
- **Shift:** Dash forward.

## Future Updates

## Installation
1. Clone the repository from GitHub: `git clone https://github.com/whahn1983/Tech-Flow-Game.git`
