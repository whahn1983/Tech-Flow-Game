# Tech Flow Runner

Welcome to the Tech Flow Runner game! This game is part of the Tech Flow Game project.

## Overview

Tech Flow Runner is an engaging and interactive game where players navigate through various challenges to collect coins and avoid obstacles. The game features interactive gameplay mechanics and visually appealing graphics that enhance the user experience.

## How to Play

1. **Start the Game:**
   - Open the `index.html` file in a web browser to launch the game.

2. **Controls:**
   - Use the arrow keys to move the character.
   - Collect coins to earn points.
   - Avoid obstacles to maintain your health.

3. **Objective:**
   - The goal is to collect as many coins as possible while avoiding obstacles. Try to beat your high score!

## Features

- Interactive gameplay with responsive controls.
- Attractive graphics and game design.
- High score tracking to encourage competition among players.

## Installation

You can download or clone the repository to run the game locally.

```bash
git clone https://github.com/whahn1983/Tech-Flow-Game.git
```

Navigate to the directory and open `index.html` in your web browser.
